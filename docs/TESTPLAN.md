# Student Account Management System Test Plan

This test plan covers all business logic implemented in the COBOL application. It is designed to validate the system with business stakeholders and will be used as a basis for future unit and integration tests in a Node.js app.

| Test Case ID | Test Case Description                | Pre-conditions                | Test Steps                                                                 | Expected Result                                      | Actual Result | Status (Pass/Fail) | Comments                |
|--------------|--------------------------------------|-------------------------------|----------------------------------------------------------------------------|------------------------------------------------------|--------------|--------------------|-------------------------|
| TC01         | View initial balance                 | Account initialized           | 1. Start app<br>2. Select 'View Balance'                                   | Balance displayed as 1000.00                         |              |                    |                         |
| TC02         | Credit account with valid amount     | Account initialized           | 1. Start app<br>2. Select 'Credit Account'<br>3. Enter 200.00              | Balance updated to 1200.00<br>Success message shown  |              |                    |                         |
| TC03         | Debit account with valid amount      | Account initialized           | 1. Start app<br>2. Select 'Debit Account'<br>3. Enter 300.00               | Balance updated to 700.00<br>Success message shown   |              |                    |                         |
| TC04         | Debit account with insufficient funds| Account balance < debit amount| 1. Start app<br>2. Select 'Debit Account'<br>3. Enter 2000.00              | Error message: 'Insufficient funds for this debit.'   |              |                    |                         |
| TC05         | Invalid menu choice                  | App running                   | 1. Start app<br>2. Enter invalid menu option (e.g., 5)                      | Error message: 'Invalid choice, please select 1-4.'   |              |                    |                         |
| TC06         | Exit application                     | App running                   | 1. Start app<br>2. Select 'Exit'                                            | App exits gracefully<br>Exit message shown            |              |                    |                         |
| TC07         | Multiple operations sequence         | Account initialized           | 1. Start app<br>2. Credit 100.00<br>3. Debit 50.00<br>4. View Balance      | Balance reflects all operations correctly             |              |                    |                         |
| TC08         | Credit with zero amount              | Account initialized           | 1. Start app<br>2. Select 'Credit Account'<br>3. Enter 0.00                | Balance unchanged<br>Success message shown            |              |                    |                         |
| TC09         | Debit with zero amount               | Account initialized           | 1. Start app<br>2. Select 'Debit Account'<br>3. Enter 0.00                 | Balance unchanged<br>Success message shown            |              |                    |                         |
| TC10         | View balance after credit/debit      | Account modified              | 1. Start app<br>2. Perform credit/debit<br>3. Select 'View Balance'         | Balance reflects latest transaction                   |              |                    |                         |
